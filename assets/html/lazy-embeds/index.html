Third party embeds lazy load

Goal

The script will lazy load the script coming with the embeded rich media. The Intersection Observer API will be used to set the source of the script, based on the scroll position.

HTML Required

To be able to target the script coming with the third party, this one must have a wrapper, with a specif class. This task will be done by a WordPress filter, as this example.
This class will be used during the initialization of the script.

And with PHP the src of the script must be changed to data-lazy-src
An example of a WordPress filter to achieve this task is available to download.
